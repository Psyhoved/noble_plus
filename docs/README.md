## Суть проекта

Мы разрабатываем мод **"Noble Plus"** (`mod_noble_plus`) для игры **Battle Brothers**. Мод строится поверх мода **Legends (версия 19.1.47)** как основной зависимости.

### Идея и видение

Мод добавляет новое происхождение "Дворянин +" — расширенную версию ванильного происхождения Disowned Noble (DLC8). Нарратив: игрок управляет опальным дворянином, изгнанным из родового замка, который пытается вернуть своё положение силой оружия.

**Глобальная цель** — двигаться в сторону уникального сюжетного опыта с механиками в духе Crusader Kings: владение замками, династии, браки, передача земель. Но это далёкое будущее. Сейчас работаем итеративно, малыми шагами.

### Что делает происхождение "Дворянин +"

По сравнению с ванильным Disowned Noble, наше происхождение добавляет три вещи:

Во-первых, дворянин стартует с **легендарным белым волком** (`legend_white_wolf_item`) в accessory slot — это самый мощный компаньон-животное в Легендах (стоимость 6000 золота, энтити `legend_white_warwolf`). Нарративно это родовой зверь благородного дома.

Во-вторых, слуга в отряде стартует с **кинжалом** в качестве дополнительного оружия (в ваниле слуга стартует с пустыми руками).

В-третьих, добавляется **механика конца игры при гибели дворянина** — аналогично происхождению Lone Wolf в Легендах. Если персонаж с флагом `IsPlayerCharacter` погибает, игра заканчивается поражением. Это отражено в описании сценария.

## Инструменты агента в этом проекте

Агент имеет доступ к следующим MCP-инструментам:

**MCP Filesystem** — чтение и запись файлов в разрешённых директориях: `D:\work`, `D:\SteamLibrary\steamapps\common\Battle Brothers`, `C:\Users\Aleksander\Documents\Battle Brothers`, `C:\Users\Aleksander\AppData\Roaming\Claude`.

**BB MCP Server** (`battle_brothers`) — три инструмента: `launch_game` запускает игру, `kill_game` завершает процесс, `read_game_log` читает и парсит `log.html` с фильтрацией по типу (only_errors, only_squirrel).
