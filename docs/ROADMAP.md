### Этап 3.4 — Система амбиций Главы I ⏳ TODO (СЛЕДУЮЩИЙ ШАГ)

**Это переход к полноценному моддингу.** Амбиции — главный механизм, которым мы двигаем игрока по сюжету в открытом мире BB. Нарратив и последовательность амбиций описаны в NARRATIVE.md, раздел "Часть II: Глава I".

**Четыре последовательные амбиции:**

| # | Амбиция | Условие завершения | Нарративный смысл |
|---|---------|-------------------|-------------------|
| 1.1 | "Нанять людей" | 8+ бойцов в ростере | Барон собирает армию |
| 1.2 | "Заработать золото" | 2000+ крон | Финансирование похода |
| 1.3 | "Найти союзников" | Союз с одним знатным домом | Политическая подготовка |
| 1.4 | "Вернуть родовой замок" | Победа над узурпатором | Финал Главы I |

**Архитектура реализации:**

Создать отдельный файл `noble_plus_ambitions.nut` — модульный файл системы амбиций.
Зарегистрировать его через MSU: панель настроек в меню мода (включить/выключить отдельные амбиции).
Амбиции активируются последовательно: каждая следующая открывается после завершения предыдущей.
Состояние прогресса сохраняется в World.Flags: `NoblePlus.Ambition.CurrentChapter`, `NoblePlus.Ambition.CurrentStage`.

Изучить паттерн амбиций в Legends (файлы `scripts/ambitions/`) и в ванили.
Изучить MSU Settings API для добавления настроек мода — образец: `mod_clever_recruiter`.

Цель: в игре последовательно появляются 4 сюжетных амбиции, каждая с нарративным текстом.
